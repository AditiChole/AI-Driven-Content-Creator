# AI-Driven Personalized Content Creator

A powerful AI-driven platform for generating **personalized content across multiple formats** using Google Gemini AI. 
This system helps users create professional, engaging, and contextually relevant content for LinkedIn, ads, emails, and conversational drafts.

# Feature
Multiple Content Types
  - LinkedIn Posts – Professional social media content
  - Ad Content – Persuasive advertising copy
  - Professional Emails – Business communication
  - Conversation Drafts – Natural conversational text

Smart Content Generation
  - Advanced prompt engineering for each content type
  - Customizable tone and style
  - Real-time content generation

Minimal UI
  - Simple, clean Streamlit interface
  - Input fields for prompt, tone, audience, and length
  - Instant preview of AI-generated content

# Prerequisites

- Python 3.9 or higher
- Google Gemini API Key (Get from [Google AI Studio](https://aistudio.google.com)
